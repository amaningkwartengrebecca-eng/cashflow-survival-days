# 🐙 Cashflow Survival Days – "Duolingo for SME Cashflow"

**Live Demo:** [https://amaningkwartengrebecca-eng.github.io/cashflow-survival-days/](https://amaningkwartengrebecca-eng.github.io/cashflow-survival-days/)  


A gamified, interactive cashflow management tool built for UK SMEs. Tested with real founders and designed to turn cashflow from a spreadsheet nightmare into a daily habit.

---

## 🚀 Features
- **Survival Days Dashboard** – See your cash runway in days (colour‑coded with avatar feedback).
- **What‑If Scenario Slider** – Simulate the impact of client payment delays (0/14/30 days).
- **Invoice Timeline (Tree Map)** – Group invoices by month, expand to see details.
- **UK Statutory Interest Calculator** – Calculates 8% + BoE base rate on overdue invoices.
- **Gamified Quizzes** – Earn XP, hearts, and coins (with heart regeneration).
- **Dark / Light Mode** – Jellyfish logo changes colour with the theme.
- **Google Sheets Sync** – Import invoices from a published Google Sheet (CSV).
- **No Backend – Works Offline** – All data stored in your browser's localStorage (MVP).

---
